The Home Lab Journey

~/blog/homelab

One of the best investments I've made in my cybersecurity journey has been building a home lab. Having a controlled environment where you can safely experiment with security tools, practice attack techniques, and test defensive measures is invaluable.


In this post, I'll walk through my current setup using Proxmox VE as the hypervisor, along with the various VMs and containers I run for different security scenarios.

Hardware Setup

~/blog/homelab/hardware

My current lab runs on a refurbished Dell OptiPlex with 32GB RAM and a 1TB SSD. For most security practice, you don't need expensive hardware - what matters more is having enough RAM to run multiple VMs simultaneously.


Key considerations when building your lab:

  • Minimum 16GB RAM (32GB recommended for multiple VMs)
  • SSD storage for better VM performance
  • Multiple network interfaces for isolated lab networks
  • CPU with virtualization support (VT-x/AMD-V)
